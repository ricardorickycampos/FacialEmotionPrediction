# Facial_Expression_Recognition_Model

How to Install:
Python 3.13
 1. Install requirements
```bash
pip install -r requirements.txt
```
2. Run
```bash
python main.py
```
## Program Description
This program is used to recognize facial expressions. 
While running, you can see the video feed and the predicted expression. 

7 expressions are recognized. [angry, disgust, fear, happy, neutral, sad]
